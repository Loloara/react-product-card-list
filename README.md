## Quick Start

### Installation

#### 환경 구축

```
Ubuntu Server v18.04 LTS (HVM)
node v10.22.0
yarn v1.22.5
typescript v4.0.3
```

#### 코드 빌드 및 실행

```
git clone https://github.com/Loloara/react-product-card-list.git
cd react-product-card-list
yarn install
yarn build
yarn start
```
